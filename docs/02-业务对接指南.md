# FreeMail 业务对接指南

本文档说明如何将你的其他业务系统与 FreeMail 对接，实现邮件接收、转发和读取。

---

## 方案总览

| 对接方式 | 适用场景 | 复杂度 |
|---------|---------|--------|
| API 轮询 | 注册验证码、邮件通知接收 | 低 |
| 邮件转发 | 将特定邮件实时转发到你的主邮箱 | 低 |
| Webhook（需扩展） | 收到邮件后实时回调你的服务 | 中 |

---

## 方案一：API 轮询（推荐）

最通用的方案。你的业务系统通过 API 创建临时邮箱，然后轮询查询收到的邮件。

### 完整流程

```
你的业务系统                          FreeMail
    │                                   │
    │── 1. POST /api/generate ─────────►│ 创建临时邮箱
    │◄─────── 返回 email 地址 ──────────│
    │                                   │
    │── 2. 用该邮箱去注册第三方 ────────►│ (第三方发送验证邮件)
    │                                   │
    │── 3. GET /api/emails?mailbox=xxx ─►│ 轮询邮件列表
    │◄─────── 返回邮件列表 ─────────────│
    │                                   │
    │── 4. GET /api/email/{id} ────────►│ 获取邮件详情
    │◄─────── 返回验证码/内容 ──────────│
    │                                   │
    │── 5. DELETE /api/email/{id} ─────►│ 用完清理（可选）
    │                                   │
```

### 认证配置

所有 API 调用需要携带 JWT Token（通过 `wrangler secret put JWT_TOKEN` 设置的值）：

```bash
# 方式 1（推荐）
Authorization: Bearer <YOUR_JWT_TOKEN>

# 方式 2
X-Admin-Token: <YOUR_JWT_TOKEN>

# 方式 3
?admin_token=<YOUR_JWT_TOKEN>
```

### Python 对接示例

```python
import requests
import time

class FreeMailClient:
    def __init__(self, base_url, jwt_token):
        self.base_url = base_url.rstrip('/')
        self.headers = {
            "Authorization": f"Bearer {jwt_token}",
            "Content-Type": "application/json"
        }

    def generate_email(self, length=8, domain_index=0):
        """生成随机临时邮箱"""
        resp = requests.get(
            f"{self.base_url}/api/generate",
            params={"length": length, "domainIndex": domain_index},
            headers=self.headers
        )
        resp.raise_for_status()
        return resp.json()["email"]

    def create_email(self, local_part, domain_index=0):
        """创建指定前缀的邮箱"""
        resp = requests.post(
            f"{self.base_url}/api/create",
            json={"local": local_part, "domainIndex": domain_index},
            headers=self.headers
        )
        resp.raise_for_status()
        return resp.json()["email"]

    def get_emails(self, mailbox):
        """获取邮箱的邮件列表"""
        resp = requests.get(
            f"{self.base_url}/api/emails",
            params={"mailbox": mailbox},
            headers=self.headers
        )
        resp.raise_for_status()
        return resp.json()

    def get_email_detail(self, email_id):
        """获取邮件详情（含完整 HTML/Text 和验证码）"""
        resp = requests.get(
            f"{self.base_url}/api/email/{email_id}",
            headers=self.headers
        )
        resp.raise_for_status()
        return resp.json()

    def wait_for_email(self, mailbox, timeout=120, interval=5):
        """等待邮件到达，返回第一封新邮件"""
        start = time.time()
        while time.time() - start < timeout:
            emails = self.get_emails(mailbox)
            if emails:
                return emails[0]
            time.sleep(interval)
        raise TimeoutError(f"等待 {timeout}s 后未收到邮件")

    def get_verification_code(self, mailbox, timeout=120):
        """等待并提取验证码"""
        email = self.wait_for_email(mailbox, timeout)
        detail = self.get_email_detail(email["id"])
        code = detail.get("verification_code")
        if code:
            return code
        # 系统已自动提取验证码，如果为空说明邮件中没有明显的验证码
        return None


# === 使用示例 ===

client = FreeMailClient(
    base_url="https://mail.codewith.site",
    jwt_token="your_jwt_token_here"
)

# 场景：自动获取第三方注册验证码
email = client.generate_email()
print(f"临时邮箱: {email}")

# ... 在这里用 email 去第三方网站注册 ...

code = client.get_verification_code(email, timeout=60)
print(f"验证码: {code}")
```

### Node.js 对接示例

```javascript
class FreeMailClient {
  constructor(baseUrl, jwtToken) {
    this.baseUrl = baseUrl.replace(/\/$/, '');
    this.headers = {
      'Authorization': `Bearer ${jwtToken}`,
      'Content-Type': 'application/json'
    };
  }

  async generateEmail(length = 8) {
    const resp = await fetch(
      `${this.baseUrl}/api/generate?length=${length}`,
      { headers: this.headers }
    );
    const data = await resp.json();
    return data.email;
  }

  async getEmails(mailbox) {
    const resp = await fetch(
      `${this.baseUrl}/api/emails?mailbox=${encodeURIComponent(mailbox)}`,
      { headers: this.headers }
    );
    return await resp.json();
  }

  async getEmailDetail(emailId) {
    const resp = await fetch(
      `${this.baseUrl}/api/email/${emailId}`,
      { headers: this.headers }
    );
    return await resp.json();
  }

  async waitForEmail(mailbox, timeoutMs = 120000, intervalMs = 5000) {
    const start = Date.now();
    while (Date.now() - start < timeoutMs) {
      const emails = await this.getEmails(mailbox);
      if (emails.length > 0) return emails[0];
      await new Promise(r => setTimeout(r, intervalMs));
    }
    throw new Error(`等待 ${timeoutMs / 1000}s 后未收到邮件`);
  }

  async getVerificationCode(mailbox, timeoutMs = 120000) {
    const email = await this.waitForEmail(mailbox, timeoutMs);
    const detail = await this.getEmailDetail(email.id);
    return detail.verification_code || null;
  }
}

// 使用示例
const client = new FreeMailClient(
  'https://mail.codewith.site',
  'your_jwt_token_here'
);

const email = await client.generateEmail();
console.log('临时邮箱:', email);
// ... 用 email 注册 ...
const code = await client.getVerificationCode(email, 60000);
console.log('验证码:', code);
```

## 方案二：邮件转发

将发送到特定地址的邮件自动转发到你的主邮箱（Gmail、企业邮箱等），适合持续监控场景。

### 配置方式

通过环境变量 `FORWARD_RULES` 配置转发规则：

```bash
# KV 格式
echo "alert=admin@your-company.com,support=support@your-company.com,*=catch-all@gmail.com" \
  | npx wrangler secret put FORWARD_RULES

# JSON 格式
echo '[
  {"prefix":"alert","email":"admin@your-company.com"},
  {"prefix":"support","email":"support@your-company.com"},
  {"prefix":"*","email":"catch-all@gmail.com"}
]' | npx wrangler secret put FORWARD_RULES
```

### 匹配规则

| 规则 | 说明 | 示例 |
|------|------|------|
| `alert=admin@xxx.com` | 收件人以 `alert` 开头的邮件转发到 `admin@xxx.com` | `alert123@codewith.site` → `admin@xxx.com` |
| `support=support@xxx.com` | 收件人以 `support` 开头的邮件转发 | `support-ticket@codewith.site` → `support@xxx.com` |
| `*=catch-all@gmail.com` | 所有未匹配的邮件转发到兜底地址 | `random@codewith.site` → `catch-all@gmail.com` |

### 注意事项

- 转发使用 Cloudflare Email Routing 的 `message.forward()` 方法
- 转发和入库是**并行**的，邮件既会转发又会保存到 FreeMail
- 转发目标邮箱需要在 Cloudflare Email Routing 中验证（Dashboard → Email → Destination addresses）
- 如果不配置 `FORWARD_RULES` 或设置为空/`disabled`，则不转发

## 方案三：Webhook 回调（需要扩展开发）

如果你需要实时收到邮件通知，可以在 `server.js` 的 `email()` 处理器中添加 Webhook 回调。

### 扩展方式

在 `src/server.js` 的 `email()` 方法中，邮件入库成功后添加 Webhook 调用：

```javascript
// 在 "=== Email handler completed successfully ===" 之前添加
if (env.WEBHOOK_URL) {
  ctx.waitUntil(
    fetch(env.WEBHOOK_URL, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        event: 'email.received',
        mailbox: mailbox,
        sender: sender,
        subject: subject,
        preview: preview,
        verification_code: verificationCode,
        received_at: new Date().toISOString()
      })
    }).catch(e => console.error('Webhook failed:', e))
  );
}
```

然后配置 Webhook 地址：

```bash
# 在 wrangler.toml 的 [vars] 中添加
WEBHOOK_URL = "https://your-service.com/api/webhook/email"

# 或通过 secret（如果 URL 包含认证信息）
echo "https://your-service.com/api/webhook/email?key=xxx" | npx wrangler secret put WEBHOOK_URL
```

## 典型业务场景

### 场景 1：自动化注册和验证码提取

```python
# 1. 生成临时邮箱
email = client.generate_email()

# 2. 用临时邮箱注册第三方服务
register_on_third_party(email, password="xxx")

# 3. 等待并获取验证码
code = client.get_verification_code(email, timeout=60)

# 4. 完成验证
verify_registration(code)
```

### 场景 2：多业务域名隔离

通过邮箱前缀区分不同业务：

```python
# 业务 A 的邮箱
email_a = client.create_email("biz-a-user123")
# → biz-a-user123@codewith.site

# 业务 B 的邮箱
email_b = client.create_email("biz-b-order456")
# → biz-b-order456@codewith.site
```

配合转发规则，自动分流到不同业务邮箱：

```
biz-a=bizA-team@company.com,biz-b=bizB-team@company.com
```

### 场景 3：邮件通知监控

```python
# 创建固定邮箱用于接收系统告警
alert_email = client.create_email("monitor-alert")

# 在你的监控系统中配置告警发送到 monitor-alert@codewith.site

# 定时检查告警邮件
while True:
    emails = client.get_emails(alert_email)
    for e in emails:
        detail = client.get_email_detail(e["id"])
        process_alert(detail)
    time.sleep(30)
```

## API 速查

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/domains` | GET | 获取可用域名列表（无需认证） |
| `/api/generate` | GET | 随机生成邮箱（参数: `length`, `domainIndex`） |
| `/api/create` | POST | 自定义前缀创建邮箱（body: `local`, `domainIndex`） |
| `/api/emails` | GET | 获取邮件列表（参数: `mailbox`） |
| `/api/email/{id}` | GET | 获取邮件详情（含 HTML、验证码） |
| `/api/email/{id}` | DELETE | 删除单封邮件 |
| `/api/emails` | DELETE | 清空邮箱所有邮件（参数: `mailbox`） |
| `/api/mailboxes` | GET | 获取邮箱列表 |
| `/api/mailbox/{addr}` | DELETE | 删除邮箱 |

详细参数和返回值见 [API使用指南](../API使用指南.md)。
